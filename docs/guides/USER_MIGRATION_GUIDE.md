## ðŸ”„ Migration Process

### Before You Update

**IMPORTANT: Export your data first!**

1. **Open the current version** of Daddy Caddy
2. **Go to Settings** (tap Settings icon at bottom)
3. **Tap "Export Data"**
4. **Save the export file** (it will be saved to your device)
5. **Optional**: Share the file to yourself via email/cloud for backup

Your export file will be named: `daddy-caddy-export-{timestamp}.json`

---

### After Updating to v2.0

**If you exported your data:**

1. **Open Daddy Caddy v2.0**
2. **Go to Settings**
3. **Tap "Import Data"**
4. **Select your export file**
5. **Wait for import to complete** (you'll see progress)
6. **Done!** All your data is now in v2.0

**If you didn't export (fresh start):**

1. Just start using the app!
2. Your old data will not be available
3. This is fine if you want a clean start

## â“ Troubleshooting

### Export Issues

**"Export Failed" error:**
- Make sure you have rounds saved in the app
- Try closing and reopening the app
- Check device storage space

**Can't find export file:**
- Check your device's Downloads folder
- Check Files app (iOS) or File Manager (Android)
- Try exporting again and sharing via email

### Import Issues

**"Import Failed" error:**
- Verify the file is a valid export file
- Make sure the file isn't corrupted
- Try exporting again from the old version

**"Invalid file format" error:**
- You might have selected the wrong file
- Make sure you select the .json export file
- File must be from Daddy Caddy v1.x

**Some data is missing:**
- Check the export file was created successfully
- Verify all data was in the old version
- Contact support if issues persist
